About Drap's Labs

Drap's Labs originally started as Drapadons little home server for a side hobby. However, the hardware it ran on, was really slow and bad. In April 2025, Drapadon bought better server hardware to expand his hobby by allowing him to run apps that he previouly could not.

From then on, Drapadon started many projects, like tunneling the server to cloudflare, which allows all the apps on the server to be on these domains. Drapadon also invited Digytt to create/help out with various projects.

Drap's Labs really kicked off when Digytt's rplace clone, otherwise known as Digyttplace, was created. People started to draw on the canvas on the first day, which was an amazing sight to see considering the server started off as a fun private hobby.

Drap's Labs has it's own Discord server, linked at the top of this page. You can use it to get updates on projects, to suggest new projects, and just talk to the community backing Drap's Labs.
